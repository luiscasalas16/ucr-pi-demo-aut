# Demo Authentication - .NET, Blazor y Microsoft Entra External ID

Este repositorio contiene una serie de demos progresivas para mostrar cómo implementar autenticación y autorización.

La intención del proyecto es mostrar la evolución paso a paso de una aplicación básica hasta una arquitectura autenticada e integrada entre aplicación web y API.

## Branches

- `01-basic-api`: Versión inicial de un API básico utilizando ASP.NET Core Minimal APIs.
- `02-authentication-api`: Agrega autenticación y autorización al API utilizando Microsoft Entra External ID.
- `03-basic-web`: Versión inicial de una aplicación Blazor Web App sin autenticación.
- `04-authentication-web`: Agrega autenticación web utilizando Microsoft Entra External ID.
- `05-authentication-web-api`: Integra la web autenticada y el api autenticado utilizando Microsoft Entra External ID.
